КАНДИДАТ НА ПОСТ ПРЕЗИДЕНТА ШКОЛЫ


Ксения Горбаненко и её программа:
1) Организовать группу для игр с младшими классами во время переменок.
2) Организовать в классах группы поддержки, состоящие из учеников, для оказания помощи неуспевающим. Участники группы поддержки получат больше возможностей для участия в делах ученического самоуправления.
3) Приглашать специалистов разных профилей для беседы с учащимися по поводу выбора ими будущей профессии.
4) Проводить школьный КВН между классами, а также спортивные мероприятия
5) Проведение благотворительных акций
6) Создать «Почту доверия» с установкой почтового ящика, где каждый желающий может направить предложение по улучшению школьной жизни
7) Продолжить проводить праздники, наиболее распространенные в других странах, например,выпускные и рождественские балы.
8) Новые формы организации досуга (клубы по интересам), чтобы дети получали новые знания и возможности для общения.
